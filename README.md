# Sitio estático: Rizoma — Fiesta de la Primavera

Archivos añadidos para un sitio básico listo para GitHub Pages:

- `index.html` — página principal con secciones (Quiénes somos, La fiesta, Novedades).
- `css/styles.css` — estilos minimalistas, toques naturales y un acento rebelde.
- `assets/logo.svg`, `assets/hero-1.svg` — imágenes SVG de prueba.

Instrucciones rápidas para publicar en GitHub Pages:

1. Sube (commit & push) los archivos al repositorio en la rama `main`.
2. En GitHub: ve a la configuración del repositorio -> Pages.
3. En la sección "Build and deployment", elige "Deploy from a branch", selecciona la rama `main` y la carpeta `/ (root)`.
4. Guarda: GitHub generará la página en unos minutos. La URL tendrá el formato `https://<usuario>.github.io/<repo>`.

Notas:
- Todos los textos marcados como `PLACEHOLDER` deben reemplazarse antes de publicar contenido final.
- Si quieres usar un dominio personalizado, añade un archivo `CNAME` con tu dominio en la raíz.
- Puedo añadir integración con Jekyll, plantillas o un generador estático si lo deseas.
